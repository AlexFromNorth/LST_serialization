# Compact Set Serialization

**Тестовое задание:** сериализация множества целых чисел 1–300 в компактную ASCII-строку с коэффициентом сжатия **не менее 2×** относительно простой сериализации вида `"1,300,237,188"`.

---

## Описание

Дано **множество** (массив, порядок не важен) целых чисел в диапазоне **1–300**, количество элементов **5–1000**. Простейший формат — строка через запятую. Цель — получить строку **в 2 раза короче** без использования алгоритмов сжатия (gzip и т.п.), используя только **ASCII-символы**.

## Как запустить

```bash
npm install
npm run test   # Jest-тесты
npm run run    # Запуск выполнения
```
